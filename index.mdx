System Informer gives you deep visibility into every process, service, network connection, and kernel object on your Windows machine — in real time. Whether you're tracking down a resource hog, analyzing malware, debugging a crash, or building your own tools on top of the Windows Native API, System Informer has you covered.

## Get started

<Steps>
  <Step title="Download System Informer">
    Get the latest release from [systeminformer.com/downloads](https://systeminformer.com/downloads) or from the [GitHub releases](https://github.com/winsiderss/si-builds/releases). No installation required.
  </Step>
  <Step title="Run with administrator privileges">
    Right-click **SystemInformer.exe** and choose **Run as administrator** to unlock full process inspection, kernel driver features, and service management.
  </Step>
  <Step title="Explore your system">
    The **Processes** tab shows every running process with color-coded highlighting. Click any process to open its properties — threads, modules, memory maps, handles, and more.
  </Step>
  <Step title="Enable plugins (optional)">
    Go to **Options → Plugins** to enable bundled plugins for GPU monitoring, .NET inspection, network tools, and ETW-based disk analysis.
  </Step>
</Steps>
